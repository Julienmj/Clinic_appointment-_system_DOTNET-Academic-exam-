# Quick Setup Guide

## 🚀 5-Minute Setup

### 1. Open Project
- Double-click `clinic system.slnx` to open in Visual Studio

### 2. Build & Run
- Press `Ctrl+Shift+B` to build
- Press `F5` to run

### 3. Login as Manager
- Email: `manager@clinic.com`
- Password: `admin123`

### 4. Setup System
1. **Register Doctors** → Add 2-3 doctors
2. **Register Services** → Add services for each doctor
3. **Test Patient Registration** → Create a test patient account
